### 校园管理系统：复杂的后台操作场景(由 AI 生成)

**使用技术**：

1. **后端**：

   - express 框架
   - mongoDB 数据库
   - bcryptjs 加密模块
   - fs 模块
   - nodemon 模块
   - jsonwebtoken 模块

2. **前端**：
   - vue 前端框架
   - vue-router 路由
   - vuex 状态管理
   - arco-design-vue 组件库

**启动**：

1. **后端**：

   1. npm install || yarn
   2. docker compose up || docker-compose up
   3. npm run start

2. **前端**：
   1. cd project
   2. npm install || yarn
   3. npm run dev
